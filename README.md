# s3cmd-zsh-plugin
Zsh autocomplete plugin for [s3cmd](https://s3tools.org/s3cmd)
## Installation
### Oh My Zsh
Clone this repo.
```bash
cd $ZSH_CUSTOM/plugins && git clone https://github.com/FFKL/s3cmd-zsh-plugin.git
```
And add it to the plugins array in `.zshrc`
```bash
plugins=(... s3cmd)
```
